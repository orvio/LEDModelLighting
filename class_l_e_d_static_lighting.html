Detailed Description

Base class for lighting cycle execution.

Objects of this type govern when the LED is powered on.

Additional effects when the LED is powered on or when it transitions between the states are governed by effect classes.

Creates a new LEDStaticLighting object.

This lighting object will keep the output active with the configured brightness if the initial state is set to CYCLE_ON. All other states will result in the output being turned off.

The assigned pin will be configured as OUTPUT.

Parameters
ledPinnumber of the pin to be used. Arduino defines like LED_BUILTIN are allowed
brightnesssets the PWM duty cycle from 0 (off) to 255 (full brightness)
initialStateonly CYCLE_ON will result in an output
onEffectsets the effect class to use when the output is active
offToOnEffectset the effect class to use when the output state transitions from CYCLE_OFF to CYCLE_ON
onToOffEffectset the effect class to use when the output state transitions from CYCLE_ON to CYCLE_OFF
